ESP32-CAM_MJPEG2SD
Needs an SD-Card with 2 files for the webserver in /data Directory

Create an SD-Card with /data directory
Save next 2 raw files to the SD-Card in a directory /data
       https://raw.githubusercontent.com/ldijkman/async-esp-fs-webserver/master/docs/ino/ESP32-CAM_Camera_Webserver/BIN/data/MJPEG2SD.htm
       https://raw.githubusercontent.com/ldijkman/async-esp-fs-webserver/master/docs/ino/ESP32-CAM_Camera_Webserver/BIN/data/common.js
https://github.com/ldijkman/async-esp-fs-webserver/tree/master/docs/ino/ESP32-CAM_Camera_Webserver/BIN/data
and place the 2 files in the /data directory on SD-Card
insert SD-Card in Slot on ESP32-CAM
